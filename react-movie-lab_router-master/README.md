# Getting Started with create React router 

firs you need to install 
``` npm i react-router-dom ``` 


### Requirements


| Component | Renders                                   | Route           |
| --------- | ----------------------------------------- | --------------- |
| Home      | "This is the homepage"                    | /               |
| About     |this is about page                         | /About/         |
| Allmovies | A list of movies                          | /movies/        |
| oneMovie  | show one movie                            | /movies/:movie  |
